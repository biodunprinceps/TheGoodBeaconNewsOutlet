PRODUCTION DEPLOYMENT
---------------------

When ready to deploy to production:

1. Read FIRST_DEPLOY_CHECKLIST.md (REQUIRED!)
2. Read SECURITY_WARNINGS.md (REQUIRED!)
3. Change ALL default passwords
4. Configure persistent storage (Railway Volumes or S3)
5. Set up email service (SendGrid, Mailgun, etc.)
6. Configure SSL/HTTPS
7. Run migrations
8. Deploy!

Recommended platforms:
- Railway.app ($5/month credit)
- Render.com (free tier available)
- DigitalOcean App Platform
- Fly.io

See DEPLOYMENT_NOTES.md for platform-specific guides.
